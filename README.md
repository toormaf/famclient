# Famroot

A modern family connection hub built with React, Vite, Tailwind CSS, and Ant Design.

## Features

- Optimized landing page with carousel
- Modular dashboard with lazy loading
- Family connections management
- Real-time chat
- Secure document vault
- Notes with tagging
- Family location mapping
- HTTPS enabled with SSL certificate

## Tech Stack

- React 18 with TypeScript
- Vite 6 for fast builds
- Tailwind CSS for styling
- Ant Design for UI components
- React Router for navigation
- HTTPS with self-signed SSL certificate

## Development

The dev server runs on HTTPS by default at `https://localhost:5173`

```bash
npm run dev
```

Note: You may see a browser warning about the self-signed certificate. This is expected for local development.

## Build

```bash
npm run build
```

## Preview

```bash
npm run preview
```
